# Clean Touch

Site vitrine React + Vite + TypeScript pour Clean Touch.

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Notes

- Le formulaire stocke actuellement les demandes dans `localStorage`.
- La structure est prête pour une connexion future à Google Sheets, Airtable, Supabase, Firebase ou un backend custom.
